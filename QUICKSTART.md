# Быстрый старт

## Установка зависимостей

### Backend
```bash
cd backend
npm install
```

### Frontend
```bash
cd frontend
npm install
```

## Запуск

### 1. Запустите backend (в отдельном терминале)
```bash
cd backend
npm run dev
```

Backend будет доступен на `http://localhost:3001`

### 2. Запустите frontend (в отдельном терминале)
```bash
cd frontend
npm run dev
```

Frontend будет доступен на `http://localhost:3000`

## Первый вход

Откройте браузер и перейдите на `http://localhost:3000`

## Создание пользователей

1. Войдите как администратор
2. Нажмите "Создать пользователя"
3. Заполните форму:
   - Email
   - Пароль (минимум 6 символов)
   - ФИО
   - Адрес
   - Номер участка
   - Телефон
   - Роль (Пользователь или Охрана)

## Использование

### Для пользователя (жителя):
1. Войдите в систему
2. Нажмите "Заказать пропуск"
3. Заполните форму заявки
4. Просматривайте, редактируйте или удаляйте свои заявки

### Для охраны:
1. Войдите в систему с ролью "Охрана"
2. Просматривайте все заявки
3. Используйте фильтры по дате и типу транспорта

### Для администратора:
1. Войдите в систему с ролью "Администратор"
2. Создавайте новых пользователей
3. Просматривайте всех пользователей системы
